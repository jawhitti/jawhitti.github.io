Completely Impractical Syntax New operators for the 64-bit features you know and love. Four-spot variables, stockade constants, 128-bit ephemeral mingle. All the things you never asked for.

A Powerful Debugger for a Ridiculous Language Breakpoints, watch expressions, dynamic expression evaluation, COME FROM visualization. The first interactive debugger ever built for INTERCAL. It works. We're as surprised as you are.

AI-Powered Of course it has AI. Everything has AI. We have to have AI. But ours analyzes your code and mocks you for it.
