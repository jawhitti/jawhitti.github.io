Write dumb code. Faster.

Same syntax, bigger numbers

We bring you a modern reimagining of the classic 1972 language, with new 64-bit language extensions, a revamped and faster compiler, and full support for VS Code development with breakpoints and watch expressions, an AI assistant that critiques your code in real time, and 17 step-by-step tutorial programs. Completely backwards compatible with classic INTERCAL. Everything you need to write code that nobody can read, now available on winget. Join the Discord!

You've seen classic INTERCAL

DO .5 <- '?"'&"!2~.5'
~'"?'?.5~.5'$#32768"
~"#0$#65535"'"$
".5~.5"'~#1"$#2'~#3

A representative expression from pi.i, constrained by the limitations of an aging 16-bit architecture.

Now witness the power of INTERCAL64

DO ::5 <- '?"'&"!:2~::5'
~'"?'?::5~::5'$####9223372036854775808"
~"####0$####18446744073709551615"'"$
"::5~::5"'~####1"$####2'~####3

The updated architecture provides relief and design freedom for tackling larger problems.
